# Repositorio local
- Para crear un repositorio local en Git, primero entra a la carpeta de tu proyecto con cd ruta/del/proyecto  

- Luego ejecuta git init para inicializar el repositorio, crea el archivo .gitignore si lo necesitas  

- agrega los archivos con git add .

- verifica el estado con git status y finalmente guarda el primer cambio con git commit -m "Primer commit" con lo cual tu repositorio local quedar√° listo para usar.
