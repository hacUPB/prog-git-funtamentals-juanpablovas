# Repositorio remoto
  
Iniciar sesión en GitHub y crear un nuevo repositorio sin inicializarlo con README.  

Abrir la terminal y entrar a la carpeta del proyecto local con cd ruta/del/proyecto.  

Inicializar el repositorio local con git init (si aún no existe).  

Agregar los archivos al control de versiones con git add ..  

Crear el primer commit con git commit -m "Primercommit".

Copiar la URL del repositorio remoto en GitHub.  

Conectar el repositorio local con el remoto usando git remote add origin URL-del-repositorio.  

Establecer la rama principal y enviar los archivos a GitHub con git branch -M main y git push -u origin main.
